## Benefício desta Disciplina
A matéria de Laboratório de Programação é o coração da formação de um desenvolvedor. Aqui, os benefícios vão além da escrita do código:
1. **Resolução de problemas complexos** Você aprenderá a decompor grandes desafios em pequenas tarefas trataveis.
2. **"Mentalidade mão na massa:** A disciplina foca na experimentação. O erro aqui não é fracasso, mas um passo necessário para o aprendizado.
